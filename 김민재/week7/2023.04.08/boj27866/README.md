# [BoJ 27866 분해합](https://www.acmicpc.net/problem/27866)

solved.ac Bronze V

## 카테고리

문자열

## 시간복잡도

시간복잡도는 O(1)이다.

## 풀이

첫째줄의 문자열에서 둘째줄에 입력된 인덱스를 출력한다.

```python
sys.stdout.write(sys.stdin.readline()[int(sys.stdin.readline()) - 1])
```
